# 📖 About Project

AI-Chattbot-Enabled-Digital-Tiffin-Service is a full-stack web application that allows customers to browse, subscribe, and order homemade tiffin services from verified vendors. The platform provides secure authentication, vendor approval workflow, subscription plans, online ordering, payment management, and an AI chatbot powered by IBM Watson Assistant to improve customer support.

The application follows a secure REST API architecture using Spring Boot on the backend and React on the frontend.
